# CogniTron
CogniTron is a free, voice-based AI interview coach that simulates realistic interview preparation experience across any role or field.

Users practice through spoken conversations, receiving instant feedback on their responses in a judgment-free environment.

We implemented a multi-model system allowing users to switch between AI models mid-practice and upload company - specific interview data to train customized evaluation models, making prep truly personalized.

Our user evaluation statistically significant improvement between first and second interviews using our depth score metric.

We achieved an average System Usability Scale score of 75.42, indicating good usability and user satisfaction.
